# BayesWDFM

This repository provides the code needed to reproduce the analysis presented in the paper: [Macrì-Demartino, R., Egidi, L., and Torelli, N. (2026). *Bayesian weighted discrete-time dynamic models for association football prediction*. Journal of the Royal Statistical Society Series C: Applied Statistics, qlag032.](https://doi.org/10.1093/jrsssc/qlag032)
